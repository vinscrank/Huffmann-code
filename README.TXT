========================
ISTRUZIONI PER ESEGUIRE IL PROGRAMMA
========================
Nella cartella di consegna è presente:
-cartella dist: contiene il file eseguibile huffmann2.jar e il file input.csv
-cartella src: contiene il codice sorgente

Per eseguire il programma:
-Aprire il terminale e posizionarsi all'interno della cartella dist del progetto 
-eseguire la seguente istruzione:     java -jar huffmann2.jar
-l'esecuzine del programma dovrebbe essere mostrata sul terminale

Si può utilizzare un altro testo di input modificando il file /dist/input.csv
